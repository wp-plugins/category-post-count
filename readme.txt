With this plugin you can set the posts_per_page and posts_per_rss settings for individual categories.

== Description ==

With this plugin you can set the posts_per_page and posts_per_rss settings for individual categories. Does not support custom taxonomies yet.

== Installation ==

1. Upload `category-post-count` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress

== Frequently Asked Questions ==

= What is the default value of `posts_per_page` and `posts_per_rss`? =

If you didn\'t change it after WordPress install, it has to be 10. But it can be changed from Settings > Reading page of your Admin area.

= Does it have to be same values for posts and feeds? =

No, you can use different values. You can even left it empty to use default value of your blog.

== Screenshots ==

1. When adding category, you can specify post count.
2. Category listing shows each category\'s posts_per_page and posts_per_rss values.
